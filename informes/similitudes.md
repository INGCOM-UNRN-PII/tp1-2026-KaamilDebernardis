Empezando con la sintaxis general, Java es muy parecido a C, manteniendo el uso de llaves, punto y coma al final de los renglones, comentarios, y los mismo s tipos, excepto el boolean que Java tiene de forma nativa.

En Java no se usan ni #include ni #define, sino que se usa import para las librerías. Java también maneja la memoria y no tiene punteros.

printf es ahora System.out.println, y scanf es Scanner.nextInt
